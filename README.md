## 📦 目录结构说明

```
├── docs/           # 博客内容与文档
├── .gitignore      # Git 忽略文件
├── package.json    # 项目依赖与脚本
├── yarn.lock       # yarn 依赖锁定文件（如用 pnpm 可删除）
├── pnpm-lock.yaml  # pnpm 依赖锁定文件（如用 yarn 可删除）
├── package-lock.json # npm 依赖锁定文件（如用 pnpm/yarn 可删除）
├── README.md       # 项目说明
├── deploy.sh       # 部署脚本
└── ...
```

## 🛠️ 配置与常见问题
